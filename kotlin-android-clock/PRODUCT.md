# Product

## Register

product

## Users
Developers and AI agents exploring Android CLI, ADB, and agent-based coding. They require a clean, high-precision, and reliable interface to verify, test, and demonstrate Android tools, CLI commands, and automated code transformations.

## Product Purpose
A test/sample Android application to explore Android CLI and Agent-based code generation capabilities. It currently serves as a current time display app and acts as a foundation for verifying UI styling and layout polish. Success means a highly-polished, precise visual interface that rejects default template scaffolding.

## Brand Personality
Modern, High-tech, Precise. The tone is clean, technical, and deliberate, mimicking the aesthetics of advanced developer tools and terminal utilities.

## Anti-references
Default Android template apps (unstyled Jetpack Compose templates with default Material 3 purple/pink color schemes, generic text placement, and unrefined paddings).

## Design Principles
1. **Precision & Engineering Focus**: Every alignment, spacing increment, and typographic choice must feel calculated and intentional, mirroring the precision of terminal-based tools.
2. **Subtlety Over Loudness**: Avoid gratuitous decorations or loud accent colors. Let structure and contrast speak, utilizing accent colors purely for state and primary focus.
3. **Agent-Friendly Structure**: Maintain clean, idiomatic Jetpack Compose code with clear component separation and descriptive keys, making it easy for subsequent agents to parse and build upon.

## Accessibility & Inclusion
Maintain full WCAG contrast requirements (minimum 4.5:1 contrast ratio for body text). Integrate subtle and purposeful micro-animations that respect OS-level reduced motion preferences.
