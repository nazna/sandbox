# example-golang-echo

> Example golang application with Echo, sqlc and zerolog

## Environment

- WSL 2 on Windows 11
- Visual Studio Code 1.64.2
- Golang 1.17.6
- Echo v4

## Development

```shell
go run ./src/main.go
```
